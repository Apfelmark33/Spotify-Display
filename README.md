# Spotify Display

This is a beginner hardware project using an ESP32 and a small display.

The goal is to show the currently playing Spotify song and control playback with buttons.

## Idea

- Show song title and artist on a display
- Use 3 buttons:
  - previous song
  - play/pause
  - next song

## Hardware (planned)

- ESP32
- small OLED display
- 3 buttons

## Status

Just started. Learning and building step by step.
